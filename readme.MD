Catatan:

Inheritance
Inheritance (pewarisan) dalam Java adalah mekanisme di mana sebuah class (disebut subclass atau child class) dapat mewarisi properti (attribute) dan perilaku (method) dari class lain (disebut superclass atau parent class).

Abstaction
Abstraction (Abstraksi) adalah salah satu pilar utama OOP (Object-Oriented Programming) yang menyembunyikan detail implementasi internal dan hanya menampilkan fungsi atau fungsionalitas penting kepada pengguna. 

Encapsulation
Enkapsulasi dalam Java adalah konsep Object-Oriented Programming (OOP) yang menggabungkan data (variabel) dan method (fungsi) ke dalam satu unit (kelas) dan menyembunyikan detail implementasi internal dari luar.

Polymorphism
Polymorphism (Polimorfisme) dalam Java adalah konsep Pemrograman Berorientasi Objek (OOP) yang memungkinkan satu antarmuka atau metode tunggal digunakan untuk melakukan tindakan yang berbeda atau merespons dalam banyak bentuk, tergantung pada objek yang memanggilnya.

Overriding
Fungsi overriding pada Java adalah kemampuan dari subclass (kelas anak) untuk menyediakan implementasi spesifik dari sebuah method yang sudah ada di dalam superclass (kelas induk)

Object composition
Object Composition (Komposisi Objek) pada Java adalah sebuah prinsip desain berorientasi objek (OOP) yang memodelkan hubungan "HAS-A" (Memiliki), di mana sebuah kelas menyertakan referensi dari satu atau beberapa objek kelas lain sebagai variabel anggotanya.

ArrayList
ArrayList adalah kelas dalam Java yang digunakan untuk menyimpan daftar objek dengan ukuran yang dapat berubah secara dinamis. Tidak seperti array biasa, ArrayList dapat mengembang atau menyusut secara otomatis saat elemen ditambahkan atau dihapus.

Exception Handling
Exception Handling pada Java adalah mekanisme untuk menangani error saat runtime (program berjalan), seperti pembagian dengan nol atau file yang hilang.

File I/O
File I/O (Input/Output) pada Java adalah proses membaca dan menulis data dari/ke penyimpanan permanen (seperti hard drive).
